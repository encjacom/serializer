## Installation

Add Serializer to your `composer.json` file. 

```json
{
  "require": {
    "itav/serializer": "~1.2"
  }
}
```
or simply fire in console:
```
composer require itav/serializer
```

Then at the top of your PHP script require the autoloader:

```bash
require 'vendor/autoload.php';
```
